# Multiplication Adventure Game

A fun multiplication game for kids to practice times tables up to 400.

## Features

- 20 progressive difficulty levels
- Problems with products up to 400
- Streak bonus system for consecutive correct answers
- Visual feedback for correct/incorrect answers
- Progress tracking within levels
- Level completion celebrations

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/multiplication-game.git
cd multiplication-game
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm start
# or
yarn start
```

4. Open your browser and navigate to `http://localhost:3000`

## Game Mechanics

- Players progress through 20 levels of increasing difficulty
- Each level contains multiplication problems based on the level configuration
- Players earn points for each correct answer
- A streak bonus applies for consecutive correct answers
- Players must complete all problems in a level to advance

## Project Structure

- `src/components/` - React components for the game UI
- `src/utils/` - Utility functions and configuration
- `src/App.js` - Main application component
